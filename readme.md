# gulp-este-blank

Gulp task to turn your Este.js project into a clean state (without the demo related files).

## Install

```bash

npm install gulp-este-blank

```

## Config

In `gulpfile.babel.js`:

```js

import blank from 'gulp-este-blank';
blank(gulp);

```

## Usage

```bash

gulp blank

```
